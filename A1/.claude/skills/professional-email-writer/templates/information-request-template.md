# Information Request Template

## Structure
```
Subject: Request for Information: [Topic]

Dear [Recipient Name],

I hope this email finds you well. I'm writing to request information about [specific topic].

The information I need includes:
• [Information request 1]
• [Information request 2]
• [Information request 3]

This information is needed for [explain purpose/reason]. The deadline for this request is [date].

If you need any additional context or have questions about what I'm looking for, please don't hesitate to ask. I'm happy to provide more details.

Thank you for your assistance with this request.

Sincerely,
[Your Name]
[Your Title]
[Company/Department]
[Contact Information]
```

## When to Use
- When requesting data, reports, or documents
- When needing clarification on a topic
- When gathering information for a project
- When following up on outstanding information requests

## Tips
- Be specific about what information you need
- Explain why you need it
- Provide reasonable deadline
- Offer to provide additional context
- Make it easy for them to respond
